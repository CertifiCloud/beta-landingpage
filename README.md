# Certificloud Landing Page

Landing page premium para a startup `Certificloud`, com product mark `CloudStudy`, construída em `Next.js` com `App Router`, `TypeScript` e `Tailwind CSS`.

## Stack

- Next.js 14+
- TypeScript
- Tailwind CSS
- Lucide React

## Node recomendado

- Node.js 20 LTS recomendado
- Compatível com Node.js `18.18+`

## Como instalar

```bash
npm install
```

## Como rodar em localhost:3000

```bash
npm run dev
```

Depois, abra:

```text
http://localhost:3000
```

## Scripts

```bash
npm run dev
npm run build
npm run start
npm run lint
```

## Projeto

- Landing page única, focada em apresentação executiva e conversão.
- Copy em português do Brasil.
- Identidade visual escura, sofisticada e inspirada em lançamentos premium de SaaS.
- Sem backend, autenticação ou dependências visuais pagas.
